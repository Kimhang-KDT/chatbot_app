# 인사, 노무 챗봇

이 프로젝트는 인사, 노무 관련 질문을 처리하는 챗봇 입니다. 

서버 - Flask, 프론트 - 리액트 네이티브 엑스포

## 요구사항
- Node.js
  - LTS 버전
- npm
- Git
- Expo Cli
  - Expo 프로젝트 생성 및 관리 도구
  - npm install -g expo-cli로 설치
- Expo App
- Java Development Kit (JDK)
  - JDK 17
- Android Studio

## 프로젝트 생성

npx create-expo-app myapp

## 실행

npx expo start
